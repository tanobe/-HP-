## サービス概要
ゼミ内でコロナウイルスの影響から、オンラインでどうにか新入生を歓迎する必要性が生じたため、サイトの方を作成致しました。簡易的なサイトではありますが是非閲覧までして頂けると嬉しいです。

 - サーバーはロリポップのものを使用しております。
 - サイト内の問い合わせはgoogleのものを埋め込み対応しております。
 - 作成日は2020 11月7日です。
